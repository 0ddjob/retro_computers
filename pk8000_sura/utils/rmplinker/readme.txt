RMPLinker для ПК8000.
Программа для преобразования музыкальных rmp-модулей в cas файлы, которые можно загружать и проигрывать в эмуляторе b2m с использованием прилагаемого конфигурационного файла Vesta AY.cfg (его надо поместить в папку config).
Чтобы послушать музыку на реале, с использованием варианта платы Sound Tracker, адаптированного для ПК8000 Михаилом Тарасовым (Mick), cas можно преобразовать в wav с помощью msxовских утилит castools или подобных.
Автор кода проигрывателя мне неизвестен, проигрыватель адаптирован с Вектора-06ц.

RMPLinker имя_rmp_модуля [имя_файла_для_ПК8000]
При запуске в одном каталоге с программой должен находится файл RMPPlayer.bin

Возможны 2 варианта вызова программы:
1. С указанием только имени исходного rmp-модуля (см. example1.bat)
RMPLinker e-type.rmp
или
RMPLinker e-type.rmp >example1.log
В результате получим файл e-type.cas, который надо загружать bload"e-type",r (имя файла для ПК8000 - max первые 6 символов имени исходного rmp-модуля)

2. С указанием имени исходного rmp-модуля и желаемого имени файла для ПК8000 (см. example2.bat). Это нужно, когда имя rmp-модуля содержит всякие левые для ПК8000 символы.
RMPLinker l@dance1.rmp ladanc
или
RMPLinker l@dance1.rmp ladanc >example2.log
В результате получим файл ladanc.cas, который надо загружать bload"ladanc",r 
Если в имени файла для ПК8000 написать более 6 символов, то будут взяты первые 6.
Если сомневаетесь, какое имя получит программа на ПК8000, можете сделать вариант с логом и посмотреть строчку
