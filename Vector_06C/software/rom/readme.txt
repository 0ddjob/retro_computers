basic25.r0m - Интерпретатор Бейсика, входящий в базовую поставку ПО к «Вектору-06Ц» на аудиокассете. Не работает на векторе с любым адаптером z80.
basic25z.r0m - Версия работающая и на КР580 и на Z80. Адаптация к z80 - Иван Городецкий, Александр Тимошенко. При адаптации ни один оператор не изменил своего функционирования. При запуске в эмуляторе b2m в диалоге выбора файла для внешнего ПЗУ используйте фильтр All files (*.*).
bas25zay.r0m - Тоже самое, что и basic25z.r0m, только PLAY играет не через ВИ53, а через Sound Tracker (Иван Городецкий, 14.07.2009)
