Программы на Бейсике :

1) Запустить Бейсик
2) - для Basic"Микро" и Basic-PLUS - ввести директиву CLOAD"", нажать Enter,выбрать файл. Затем ввести RUN  и нажать Enter.
   - для Basic-BEST - ввести директиву MLOAD, нажать Enter,выбрать файл. Затем ввести RUN  и нажать Enter.

Программы на Паскале :
(версия Паскаля для Микроши - не прочиталась. В этом архиве приведена та-же программа, но для РК-86. т.е запускать ее нужно из режима РК-86)
1) Запустить Паскаль.
2) на вопрос "NEW?" нажать клавишу "Y"
3) нажать клавишу "PageUP" (в интерпретации Микроши и РК - это клавиша "СТР") - программа перейдет в режим редактирования.
4) нажать "Esc"+"I" - на запрос "Имя?" просто нажать "Enter" и выбрать файл.
5) после того, как текст программы появится на экране - нажать "PageUP" - включится режим Трансляции. Потом нажать "Enter" для запуска программы. Затем - стоит немного подождать (~1-2 мин) пока программа оттранслируется в машинный код и запустится.
