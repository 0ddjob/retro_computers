3.8. АЛФАВИТНЫЕ КОЛЬЦА .
Цель игры: Изображены четыре пересекающиеся кольца с латинских алфавитом. При повороте любого из них расположение букв изменяется.

Ваша задача - восстановить исходный порядок после того, как буквы автоматически перемешались.

Правила игры:

На экране высветились алфавитные кольца, после нажатия любой клавиши дается правила игры:

уровень выбираете сами нажатием клавиш "КРВВ" или "КРВН".

После этого нажимаем снова клавишу ВВОДА и входим в игру.

Левое кольцо поворачивается по часовой стрелке при нажатии 2, против часовой - при нажатии 1.

Верхнее кольцо поворачивается по часовой стрелке при нажатии 4, против часовой - при нажатии 3.

Правое - при нажатии 6 и 5.

Нижнее - при нажатии 8 и 7.

При нажатии 0 временно изображается исходная картинка.
* - прекращение игры.

9 - ход назад.

По часовой стрелке: 2,4,6,8.

Поотив часовой стрелки: 1,3,5,7.

Ваш уровень:

По прежнему результату.

Начинающий.

Любитель.

Знаток.

Мастер.

Гроссмейстер.

В случае успешной игры Вы с уровня " Начинающий " сможете подняться до уровня " Гроссмейстер " Соответственно возрастет сложность изначального " запутывания " колец, но, вместе с тем, изменится и поздравительная мелодия.

Запуск эмуляции МС1502 в MESS.
------------------------------
После запуска MESS, наберите mc1502, затем [Enter] и наберите OK.
После теста памяти, нажмите [Enter] и ещё раз [Enter]

Директивы монитора.
-------------------
Загрузка:
L GOP [Enter]

Запуск:
G [Enter]

Основные команды в эмуляторе.
-----------------------------
1. Выбор и загрузка файла
a) Нажмите [Scroll Lock] - для перехода в интерфейс
b) Нажмите [TAB] (либо как настроите) и выберите File Manager, выберите требуемый WAV файл, затем снова нажмите [TAB], либо перейдите в основное меню через выбор Return to Prior Menu.
c) Введите директиву монитора L <ИМЯ СОГЛАСНО НАЗВАНИЮ MP3 ФАЙЛА>, нажмите [Enter]
d) Нажимте [TAB] и перейдите в Tape Control, выберите Play, нажмите [TAB] для закрытия интерфейса.

2. Быстрый выход из игры по "сбросу" 
Нажмите [Shift+F3], либо [F3] (как настроите) в эмуляторе MESS.

3. Ускоренная загрузка игры 
Нажмите [F10], либо удерживайте нажатым INSERT (либо как настроите) находясь в режиме эмуляции (интерфейс закрыт).
