3.13. ТОЛМАЧ.

Цель игры - игровая обучающая программа GRIF служит для обучения

лексике иностранного (в данном случае - английского) языка.

Правила игры .

Нажимаем любую клавишу для того ,чтобы начать игру. Слева направо по экрану ползет английское слово. Из меню в верхней части экрана следует выбрать перевод. Перевод выбирается клавишами "КРВН" или "КРВВ". Если выбрали перевод, то для ответа нажимаем клавишу ПРОБЕЛ. Словарь условно разбит на несколько тем ( дом, погода, одежда, животные, путешествия). Если выбранный перевод верный, то облако с английским словом потухнет, а в гистограмме в правой верхней части экрана вырастет столбец, соответствующий данной теме. Величина роста зависит от скорости подбора перевода. Если перевод неверный, то облако с иностранным словом несколько раз мигнет, а в гистограмме в правой нижней части экрана вырастет столбец, соответствующий числу сделанных в словах этой темы ошибок.

Выход из программы осуществляется клавишей АР2.

В редакции 1.00 программы словарный запас составляет всего 400 с небольшим слов, разбитых на 6 тем.

Запуск эмуляции МС1502 в MESS.
------------------------------
После запуска MESS, наберите mc1502, затем [Enter] и наберите OK.
После теста памяти, нажмите [Enter] и ещё раз [Enter]

Директивы монитора.
-------------------
Загрузка:
L GRIF [Enter]

Запуск:
G [Enter]

Основные команды в эмуляторе.
-----------------------------
1. Выбор и загрузка файла
a) Нажмите [Scroll Lock] - для перехода в интерфейс
b) Нажмите [TAB] (либо как настроите) и выберите File Manager, выберите требуемый WAV файл, затем снова нажмите [TAB], либо перейдите в основное меню через выбор Return to Prior Menu.
c) Введите директиву монитора L <ИМЯ СОГЛАСНО НАЗВАНИЮ MP3 ФАЙЛА>, нажмите [Enter]
d) Нажимте [TAB] и перейдите в Tape Control, выберите Play, нажмите [TAB] для закрытия интерфейса.

2. Быстрый выход из игры по "сбросу" 
Нажмите [Shift+F3], либо [F3] (как настроите) в эмуляторе MESS.

3. Ускоренная загрузка игры 
Нажмите [F10], либо удерживайте нажатым INSERT (либо как настроите) находясь в режиме эмуляции (интерфейс закрыт).
