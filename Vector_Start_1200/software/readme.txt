Загрузка в Бейсик-1200 - MLOAD (без кавычек и прочего), при успешном начале загрузки на бордюре должны пойти цветные полосы

Для успешного запуска игры "Удавчик" нужно перед загрузкой Бейсика в мониторе сделать #5000 (установить границу буфера бейсика 5000h, по умолчанию 4500h)
В связи с вышесказанным запустить "Удавчик" в rom-варианте Бейсика-1200 (Сигнал) не удастся.
