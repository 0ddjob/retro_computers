Записать образ в линукс можно так:
1. Заклеить правое окошко на дискете тем самым перевести дисковод в режим 700кб.
2. Отформатировать fdformat /dev/fd0u800
3. Записать из образа дискету dd if=unix.dsk of=/dev/fd0u800
Так же можно сделать образ с дискеты dd if=/dev/fd0u800 of=obraz.dsk
Или же можете записать образ любым другим удобным для вас способом.

Инструкция по запуску:
Ставите дискету в первый дисковод!(номер 0) ВАЖНО!
Появляется приглашение загрузчика :(двоеточие)
Пишите название ядра lsx
Жмёте ввод
Ждёте появления #(решётка)
Всё система готова к работе!

Комманды:
CHDIR - сменить текущую директорию (например: CHDIR /USR)
LS - вывести содержимое дериктории (LS -L /BIN)
CP - скопировать файл
MV - переместить или переименовать
RM - удалить файл
MKDIR - создать директорию
RMDIR - удалить директорию
AS - ассемблер
LD - линкер
CC - компилатор языка C
ED - текстовый редактор
OD - octal dump
CHECK - проверка файловой системы

Можно перевести терминал в режим строчных букв коммандой STTY -LCASE только не забудте потом нажать Shift + ФИКС на клавиатуре.
Клавиша ЗБ (Backspace) чтото типа СТОП (KILL)
Вывести содержимое файла на консоль CP HELLO.C /DEV/TTY8
Можете попробовать скомпиллировать программку:
CC HELLO.C
запускать так:
./A.OUT
