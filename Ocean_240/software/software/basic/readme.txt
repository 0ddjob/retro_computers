Формат запуска:
B:MB B:DEMO

Когда внутри:
LOAD"B:DEMO"
SAVE"B:DEMO",A

Если набрать:
SAVE"B:DEMO",P
Или просто
SAVE"B:DEMO"
сохранятся каракули в формате самого этого бэйсика, потом сами же будете искать свою писанину!

LIST - распечатка программы на экране.

RUN - запуск на исполнение.
